# Tailwind CSS Learning Project

This is a learning project that demonstrates the basic usage and features of Tailwind CSS framework. The project includes a responsive landing page with various UI components and dark mode support.

## Features

- Responsive design
- Dark mode support
- Custom components using Tailwind utility classes
- Hero section with image overlays
- Article cards with hover effects
- Pagination component
- Newsletter subscription form
- Responsive navigation
- Footer with social links

## Project Structure

```
my-tailwind-project/
├── src/
│   ├── css/
│   │   ├── input.css          # Main CSS file with Tailwind imports
│   │   └── tailwind.config.css # Tailwind configuration
│   └── js/
│       └── main.js            # JavaScript entry point
├── public/
│   └── Tailwind.svg          # Site favicon
├── index.html                # Main HTML file
├── vite.config.js           # Vite configuration
├── package.json             # Project dependencies
└── README.md               # Project documentation
```

## Technologies Used

- Tailwind CSS v4.1.4
- Vite v6.3.1
- @tailwindcss/vite v4.1.4

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Components

The project includes several reusable components:

- Navigation bar with responsive design
- Hero section with gradient overlays
- Article cards with hover animations
- Pagination system
- Newsletter subscription form
- Footer with multiple columns

## Styling

All styling is done using Tailwind CSS utility classes. Custom components are defined in `src/css/input.css` using Tailwind's `@apply` directive for better reusability.
